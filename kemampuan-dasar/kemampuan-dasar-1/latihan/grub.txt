Boot loader adalah program perangkat lunak pertama yang berjalan saat komputer dijalankan. Ia bertanggung jawab untuk memuat dan mentransfer kontrol ke perangkat lunak kernel sistem operasi (seperti Linux atau GNU Mach). Kernel, pada gilirannya, menginisialisasi sisa sistem operasi (mis. Sistem GNU).

GNU GRUB adalah boot loader yang sangat kuat, yang dapat memuat berbagai sistem operasi gratis, serta sistem operasi berpemilik dengan pemuatan rantai1. GRUB dirancang untuk mengatasi kompleksitas mem-boot komputer pribadi; baik program dan manual ini terikat erat pada platform komputer itu, meskipun porting ke platform lain mungkin diatasi di masa depan.

Salah satu fitur penting dalam GRUB adalah fleksibilitas; GRUB memahami sistem file dan format yang dapat dieksekusi kernel, sehingga Anda dapat memuat sistem operasi sewenang-wenang sesuai keinginan Anda, tanpa merekam posisi fisik kernel Anda pada disk. Dengan demikian Anda dapat memuat kernel hanya dengan menentukan nama file dan drive serta partisi tempat kernel berada.

Saat mem-boot dengan GRUB, Anda dapat menggunakan antarmuka baris perintah (lihat antarmuka Baris perintah), atau antarmuka menu (lihat Antarmuka menu). Menggunakan antarmuka baris perintah, Anda mengetik spesifikasi drive dan nama file kernel secara manual. Di antarmuka menu, Anda cukup memilih OS menggunakan tombol panah. Menu ini didasarkan pada file konfigurasi yang Anda persiapkan sebelumnya (lihat Konfigurasi). Saat berada di menu, Anda dapat beralih ke mode baris perintah, dan sebaliknya. Anda bahkan dapat mengedit entri menu sebelum menggunakannya.
